## 外卖速查

### 高碳日
| 店 | 点法 |
|------|------|
| 沙县 | 鸡腿饭 + 拌面（小份）|
| 减脂餐 | 糙米加量 + 鸡胸200g + 蔬菜 |
| 黄焖鸡 | 黄焖鸡 + 大份米饭 |
| 盖浇饭 | 大份米饭 + 荤菜 |

### 中碳日
| 店 | 点法 |
|------|------|

| 店 | 点法 |

| 烤肉/牛排 | 单点肉 + 蔬菜（不要饭）|

## 计算方式

热量 = 碳水×4 + 蛋白质×4 + 脂肪×9

例：200g米饭 + 250g鸡胸 + 150g西兰花
- 碳水：52 + 0 + 6 = 58g
- 蛋白质：5 + 58 + 6 = 69g
- 脂肪：0.5 + 5 + 0.6 = 6g
- 热量：58×4 + 69×4 + 6×9 = 232 + 276 + 54 = 562 kcal
